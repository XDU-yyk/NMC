# 新飞控电机顺序/旋向/桨向现场卡

日期: 2026-07-05

用途: 新飞控到货后，在装桨和低空方向验收之前，单独确认电调接线、M1-M4 电机顺序、旋转方向和桨叶安装方向。完整流程见 `replacement_fc_ready_workflow.md`，现场总清单见 `replacement_fc_arrival_checklist.md`，证据记录见 `replacement_fc_acceptance_log_template.md`。

核心原则:

```text
全程先拆桨。
ESP32 不接电调信号线，也不修正电机顺序或方向。
电机输出、解锁、failsafe 和电调驱动由飞控/Betaflight 负责。
任何起飞瞬间翻滚、猛烈侧倾或原地打转，都先回到本卡检查。
```

## 1. 接线复核

若电调是普通白/红/黑三针舵机头:

```text
白线 -> 飞控 M1-M4 / S1-S4 信号
黑线 -> 飞控 GND
红线 -> 只在确认需要 BEC 供电时保留一根；否则退针或绝缘
```

用户 2026-07-05 提供的电调照片属于这个接线类型。现场先按 PWM/OneShot 处理，不默认 DShot；红线先按 BEC 待确认处理，未用万用表量到稳定 5V 前不要拿它给飞控供电。

电调动力线:

```text
粗红线 -> 电池/PDB 正极
粗黑线 -> 电池/PDB 负极
三根粗相线 -> 对应电机三相线
```

不要把 ESP32 GPIO 接到任意电调白色信号线。

## 3. 旋转方向记录

无桨、一次只推一个电机，低油门确认旋向:

| 电机 | Betaflight 期望旋向 | 实际旋向 | 修正动作 | 通过 |
| --- | --- | --- | --- | --- |
| M1 | CW / CCW | CW / CCW | 无 / 交换任意两根相线 | 是 / 否 |
| M2 | CW / CCW | CW / CCW | 无 / 交换任意两根相线 | 是 / 否 |
| M3 | CW / CCW | CW / CCW | 无 / 交换任意两根相线 | 是 / 否 |
| M4 | CW / CCW | CW / CCW | 无 / 交换任意两根相线 | 是 / 否 |

处理规则:

```text
三线 PWM/OneShot 电调: 旋向错时，交换该电机任意两根粗相线。
DShot/BLHeli 可用软件改向时，仍要无桨复测实际旋向。
旋向没复测通过，不装桨。
```

桨叶通过标准:

```text

桨叶正反面没有装反。
桨帽/螺母固定可靠。
装桨前已经完成 failsafe 验证。
```

## 5. 失败现象快速回退

| 现象 | 优先检查 |
| --- | --- |
| 一推油门就翻滚 | M1-M4 位置、电机旋向、桨叶方向、飞控朝向 |
| 原地疯狂偏航 | 对角电机旋向、Yaw 通道、飞控朝向、桨叶方向 |
| 只向一侧猛倾 | 单个电机输出口接错、单个电机旋向错、单个桨装错 |
| 油门很高仍不离地 | 桨叶全装反、电调油门行程、协议设置、电池电压 |
| 关闭 MC6C 后仍加油 | failsafe 未通过，禁止继续飞行 |

失败处理:

```text
立刻收油/断解锁。
断电、拆桨。
回到 Receiver、模型预览、Motor Order、Motor Direction、桨叶安装方向逐项复查。
不要通过 ESP32 代码补偿任何电机/桨/飞控朝向错误。
```
