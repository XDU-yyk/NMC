
### 项目综合总结（2026-07-03 更新）

本项目为 **"智能无人机伞"原型**，采用 **ESP32-S3（协处理器）+ F4V3S PLUS（飞控）** 双处理器架构，构建近距离辅助跟随与态势感知演示平台。

#### 当前阶段：无飞控保底演示

- **F4V3S PLUS 飞控已损坏**，当前暂停所有飞行、解锁、电机测试、MSP 控制写入和自主辅助输出。
- 当前聚焦 **ESP32-S3 伴随计算机的传感器/Web 展示能力**：
  - **VL53L1X ToF** 前向激光测距（I²C，GPIO41/42）
  - **NEO-M8N GPS** 定位诊断（UART）
  - **OV2640/OV5640 摄像头** Web 预览（独立固件，约 5 FPS）
  - **WiFi AP Web 界面**：实时传感器状态、参数调节、日志显示
- 飞控集成已暂停，待替换硬件后恢复 UART3 MSP 通信恢复路线。

#### 定位声明

> **近距离辅助跟随与态势感知原型** — ESP32-S3 伴随传感与 Web 演示平台。
> 不声称当前具备飞行能力、高精度自主人体跟随、复杂户外自主飞行或 UWB 定位。

### 补充说明
- **UWB** 不是当前硬件，仅作为未来扩展方向。
- **ESP32-S3 不直接控制电机 PWM**，仅通过 MSP/UART 向飞控发送模式/辅助指令。当前飞控损坏，此路径暂停。
- 摄像头仅用于 Web 预览/调试，**不用于飞行闭环视觉控制**。
- 当前演示价值：展示 ESP32-S3 作为无人系统协处理器的传感器读取、Web 交互和参数调试能力。
