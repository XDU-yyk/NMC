# 低空手动方向验收故障排查卡

日期: 2026-07-05

用途: 新飞控、MC6C、无桨电机顺序/旋向和 failsafe 都通过后，第一次低空手动方向验收若出现异常，按本卡回退排查。完整流程见 `replacement_fc_ready_workflow.md`。

核心原则:

```text
异常先收油/断解锁。
回到台架前先拆桨。
手动方向问题必须在 Betaflight、接收机、飞控朝向、电机/桨方向里解决。
不要用 ESP32 代码补偿 MC6C 手动飞行方向错误。
```

## 1. 立即停飞条件

出现任一现象，立刻停止低空验收:

- 一推油门就翻滚。
- 机体猛烈侧倾或原地快速打转。
- 油门收下后电机仍明显加速。
- AIL / ELE / RUD 任一方向和 MC6C 操作相反。
- 关闭 MC6C 后没有进入 failsafe。
- 飞手无法稳定收油和断解锁。

停飞动作:

```text
收油 -> 断解锁 -> 断动力电 -> 拆桨 -> 回到对应排查项。
```

## 2. 症状快速定位

| 现象 | 优先排查 | 不要先做 |
| --- | --- | --- |
| 起飞瞬间翻滚 | M1-M4 位置、电机旋向、桨叶方向、飞控朝向 | 不要改 ESP32 |
| 原地快速自旋 | 对角电机旋向、桨叶方向、Yaw 通道、飞控朝向 | 不要加大油门试 |
| 向固定一侧猛倾 | 单个电机输出口、单个电机旋向、单个桨装反、机架重心 | 不要装桨继续试 |
| 油门很高仍不离地 | 桨全装反、电调协议/校准、电池电压、桨型号 | 不要继续高油门烧电调 |
| AIL 左右反了 | Receiver Roll 方向、MC6C AIL 反向拨码、Betaflight 模型预览 | 不要用 Web 修正 |
| ELE 前后反了 | Receiver Pitch 方向、MC6C ELE 反向拨码、模型预览机头俯仰 | 不要只看数值增减 |
| RUD 左右反了 | Receiver Yaw 方向、MC6C RUD 反向拨码、飞控朝向、桨向 | 不要混改电机和遥控 |
| 悬停大幅漂移 | 加速度计校准、飞控安装水平、重心、风、桨/电机状态 | 不要直接启用 ESP32 辅助 |
| 关闭遥控后仍加油 | Failsafe 配置、接收机失控输出、ARM/Modes 设置 | 不要继续飞 |

## 3. 回退排查顺序

按顺序来，不跳:

1. 拆桨。
2. 只接 USB，打开 Betaflight Setup 页面。
3. 手动转动机架，模型预览方向必须一致。
4. Receiver 页面确认 AIL/ELE/THR/RUD/CH5/CH6 六路都动。
5. 重新记录 Channel Map，若不是 `AETR1234`，写清最终实际 map。
6. Motors 页面无桨逐个推电机，核对 M1-M4 位置。
7. 无桨核对每个电机旋向。
8. 核对每个桨叶型号、正反面和安装位置。
9. 再做 failsafe。
10. 全部通过后，才允许重新低空手动方向验收。

## 4. 通道反向处理

Receiver 页面方向不对时，优先顺序:

```text
Betaflight Channel Map
MC6C AIL/ELE/THR/RUD 反向拨码
Receiver Provider / 输入协议
```

注意:

- ELE 前推时，最终以 Betaflight 模型预览“机头下压、向前飞”为准；不同接收机数值增减可能不同。
- 不要打开 MC6C 的 `V-TAIL` 或 `ELEVON` 来修多旋翼方向。
- 不要在 ESP32 里把反向通道硬改成“看起来正确”。

## 5. 电机和桨处理

电机顺序不对:

```text
先改电调白色信号线到正确 M1-M4/S1-S4 输出口。
若 Betaflight 支持资源重映射，也必须重新无桨逐个验证。
```

电机旋向不对:

```text
普通三线 PWM/OneShot 电调: 交换该电机任意两根三相线。
DShot/BLHeli 改向后也必须无桨复测。
```

桨叶不对:

```text
先确认电机旋向，再按该旋向装对应正桨/反桨。
桨叶正反面必须正确，不能只看左右位置。
```

## 6. 重新验收前检查

重新低空方向验收前，必须再次确认:

- `replacement_fc_motor_direction_card.md` 已更新。
- `replacement_fc_acceptance_log_template.md` 已记录失败现象和修正动作。
- `failsafe_video.mp4` 仍然有效。
- CH6 保持低位，ESP32 不参与第一次手动方向飞行。
- 飞手站在机尾后方，机头朝空旷方向。

通过标准仍然是:

```text
油门能稳定增升力并能稳定收油。
AIL 左/右、ELE 前/后、RUD 左/右都和 MC6C 操作一致。
没有起飞瞬间翻滚、猛烈侧倾或原地打转。
```
